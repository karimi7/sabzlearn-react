## 🛠️ Tech Stack

- **Frontend**: React, Vite
- **Package Manager**: pnpm
- **Version Control**: Git

## 🚀 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) and [pnpm](https://pnpm.io/) installed.

### Installation

cd frontend

pnpm install

pnpm dev

**Clone the repository:**

```bash
git clone https://github.com/karimi7/sabzlearn-react.git
```
